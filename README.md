# Warren Gill - Personal Portfolio

Personal portfolio website showcasing professional experience, resume, and cover letter.

## Structure

- `index.html` - Landing page with hero section and navigation
- `resume.html` - Detailed resume with expandable sections
- `cover-letter.html` - Cover letter page
- `styles.css` - Site-wide styles
- `script.js` - Main JavaScript
- `cover-letter.js` - Cover letter functionality
- `_default.php` - PHP configuration

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Merriweather, Roboto)

## Local Development

Open `index.html` in a browser or serve with any static file server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`
